Typography

Font size can be adjusted using the 0-9 rem scale.

  • fs-1
  • fs-2
  • fs-3
  • fs-4
  • fs-5
  • fs-6
  • fs-7
  • fs-8
  • fs-9

While this looks pretty cool, some of these sizes serve minimal purpose.

That's why you can also use h1-h6 sizes, as well as the size of body text.

For example, you can make an h1 have the same font size as an h6, but stay semantically correct. 

<h1 class="fs-h6">Header 1</h1>

Header 1

Or set it to the same size as paragraph (body) text,

<h1 class="fs-body">Header 1</h1>
