Border

A 1px border can be added effortlessly using .b- then a color.

Example:

class="bg-blue white b-black py-5 px-3"
This has a border

Border Radius

Using the utility class .br- then a size, we can control the border radius of an element.

Example:

class="bg-blue white br-4 py-5 px-3"
This has a border radius

You can also apply the class .br-circle for a circular border radius.

Example:

class="bg-blue white br-circle py-5 px-3"

Circular border radius

Box Shadow

Add a shadow to an element using .bs- then a size of the blur.

Example:

class="bg-blue white br-4 bs-4 py-5 px-3"
This has a box shadow
